# Workout Buddy

A MERN stack app for tracking workouts with JWT authentication.

![Workout Buddy Screenshot](https://i.ibb.co/s3nXrzs/Screenshot-73.png)

## Setup

1. Download the code.

2. Install dependencies.

3. Create a `.env` file with:
   - `MONGODB URI`
   - `PORT`
   - `SECRET`

4. Run `npm run dev`.
